## URL 검증 기준

✅ **유효한 URL 기준:**
- 기관 공식 홈페이지 (404 에러 없음)
- 접근 가능한 페이지
- 신뢰도 높은 기관
- 실제 존재하는 URL

❌ **제외할 URL:**
- 404 에러 페이지
- 존재하지 않는 페이지
- 임의로 생성된 URL
- 검증되지 않은 링크
