## APA 방식 출처 포맷 (7판 기준)

### 기본 구조
```
Author(s). (Year). Title of work. Retrieved from URL
```

## 출처 링크 기재 원칙

1. **저자/기관명 명시** - 약자가 아닌 정식 기관명 사용
2. **발행연도 명시** - (Year, Month) 형식
3. **제목 명시** - 정확한 보고서/논문 제목
4. **URL 명시** - 직접 접근 가능한 URL
5. **접근일자** - 필요시 "Retrieved from" 다음에 표기

## 신뢰도 강화 요소

- ✅ APA 방식 정식 출처 기재
- ✅ 정확한 기관명 (약자 아님)
- ✅ 발행연도 및 월 명시
- ✅ 정확한 보고서 제목
- ✅ 직접 접근 가능한 URL
- ✅ 공신력 높은 기관/학술지만 사용
