# Imweb iframe 코드 - ESG EXPORT INSIGHT

## 📋 Imweb에 복사할 HTML 코드

다음 코드를 Imweb의 **HTML 삽입** 또는 **커스텀 HTML** 섹션에 붙여넣으세요:

```html
<!-- ESG EXPORT INSIGHT - Imweb iframe -->
<div style="width: 100%; max-width: 1200px; margin: 0 auto;">
    <iframe 
        src="https://jennifersujinseo.github.io/AI_TRADE_REPORT/?v=latest" 
        style="
            width: 100%;
            height: 1200px;
            border: none;
            border-radius: 12px;
            box-shadow: 0 8px 24px rgba(0, 0, 0, 0.1);
            display: block;
            margin: 20px 0;
        "
        title="ESG EXPORT INSIGHT - AI & Global Trade 심층 분석"
        allow="fullscreen"
        loading="lazy">
    </iframe>
</div>

<style>
    @media (max-width: 768px) {
        iframe {
            height: 900px !important;
        }
    }
</style>
```

## ✅ 설치 방법

### Imweb에 설치하기:

1. Imweb 관리자 페이지 접속
2. 해당 페이지 편집
3. **HTML 삽입** 또는 **커스텀 코드** 섹션 선택
4. 위의 iframe 코드 복사 & 붙여넣기

- 별도 설치 불필요
- 자동 업데이트 (매일 자정)

## 📞 문제 해결

**iframe이 로드되지 않는 경우:**
1. 브라우저 캐시 삭제 (Ctrl+Shift+Delete)
2. 페이지 새로고침 (Ctrl+F5)
3. 다른 브라우저에서 테스트

**높이 조정이 필요한 경우:**
- `height: 1200px` 값을 원하는 높이로 변경
